# 数据字典

## 1. 字段命名原则

所有内部字段先统一使用英文 snake_case，页面展示时再翻译为中文。

## 2. 股票主数据 `securities`

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| `symbol` | string | 内部统一股票代码 |
| `market` | string | `cn` 或 `hk` |
| `exchange` | string | 交易所，例如 `sse`、`szse`、`hkex` |
| `name` | string | 股票名称 |
| `list_date` | date | 上市日期 |
| `status` | string | 正常、停牌、退市等 |
| `is_st` | bool | A股 ST / *ST 标记 |
| `industry` | string | 行业 |

## 3. 交易日历 `trade_calendar`

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| `trade_date` | date | 交易日 |
| `market` | string | `cn` 或 `hk` |
| `is_open` | bool | 是否开市 |
| `data_status` | string | `ok`、`warning`、`failed`、`stale` |

说明：MVP 已接 A股交易日历；港股交易日历先允许用实际行情日期反推，后续补稳定免费源后再固化。

## 4. 日线行情 `daily_bars`

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| `trade_date` | date | 交易日 |
| `symbol` | string | 股票代码 |
| `market` | string | 市场 |
| `open` | float | 开盘价 |
| `high` | float | 最高价 |
| `low` | float | 最低价 |
| `close` | float | 收盘价 |
| `volume` | float | 成交量，内部统一为股数 |
| `volume_unit` | string | 内部成交量单位，第一版固定为 `share` |
| `source_volume_unit` | string | 数据源原始成交量单位，例如 A股 AkShare 日线为 `lot` |
| `amount` | float | 成交额 |
| `turnover_rate` | float | 换手率 |
| `adj_factor` | float | 复权因子，可缺省 |

## 5. 指数行情 `index_bars`

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| `trade_date` | date | 交易日 |
| `symbol` | string | 指数代码，例如 `000300.CN` |
| `market` | string | `cn` 或 `hk` |
| `open` | float | 开盘点位 |
| `high` | float | 最高点位 |
| `low` | float | 最低点位 |
| `close` | float | 收盘点位 |
| `volume` | float | 成交量，按数据源原始单位保留 |
| `amount` | float | 成交额 |
| `turnover_rate` | float | 换手率 |
| `pct_chg` | float | 涨跌幅百分比 |
| `change` | float | 涨跌点数 |

说明：MVP 先接 A股指数日线，供相对指数收益、市场状态和回测基准使用；港股指数历史行情需另补稳定免费源。

## 6. 估值快照 `valuation_snapshots`

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| `trade_date` | date | 快照日期 |
| `symbol` | string | 股票代码 |
| `market` | string | `cn` 或 `hk` |
| `last_price` | float | 最新价/收盘后参考价 |
| `pe_ttm` | float | 动态市盈率或可用市盈率字段 |
| `pb` | float | 市净率 |
| `dividend_yield` | float | 股息率，可缺省 |
| `total_market_cap` | float | 总市值 |
| `float_market_cap` | float | 流通市值，可缺省 |
| `amount` | float | 成交额 |
| `turnover_rate` | float | 换手率 |

说明：估值快照来自免费行情快照字段，第一版只用于股票池过滤和解释辅助，不作为严格财报因子。

## 7. 因子表 `factor_values`

| 字段 | 类型 | 说明 |

## 8. 策略信号 `strategy_signals`

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| `trade_date` | date | 信号日期 |
| `strategy_id` | string | 策略 ID |
| `symbol` | string | 股票代码 |
| `market` | string | 市场 |
| `score` | float | 综合分数，0-100 |
| `rank` | int | 策略内排名 |
| `risk_score` | float | 风险分数，0-100，越高风险越大 |
| `risk_level` | string | 低、中、中高、高 |

| `reason_summary` | string | 小白解释摘要 |

## 9. 股票池过滤输出 `universe_filter_decisions`

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| `symbol` | string | 股票代码 |
| `market` | string | `cn` 或 `hk` |
| `is_eligible` | bool | 是否进入候选股票池 |
| `reasons` | json | 排除原因，例如 `st_stock`、`suspended`、`new_listing`、`low_price`、`low_liquidity` |

第一版过滤规则只输出结构化原因，不直接影响策略评分阈值；具体阈值由主控确认后固化。

## 10. 数据源元信息字段

以下字段由数据线程统一补充，便于后续排查免费数据源的稳定性和字段口径。

| 字段 | 类型 | 适用表 | 说明 |
| --- | --- | --- | --- |
| `source` | string | 所有原始/处理后数据 | 数据源名称，例如 `akshare` |
| `source_function` | string | 所有原始/处理后数据 | 来源接口，例如 `stock_zh_a_hist` |
| `ingested_at` | datetime | 所有原始/处理后数据 | 本地抓取入库时间 |
| `raw_symbol` | string | `securities`、`daily_bars`、`index_bars`、估值快照 | 数据源原始代码 |
| `currency` | string | `daily_bars`、估值快照 | 价格币种，A股为 `CNY`，港股为 `HKD` |
| `amount_currency` | string | `daily_bars`、估值快照 | 成交额币种，A股为 `CNY`，港股为 `HKD` |
| `adjust_type` | string | `daily_bars` | `none`、`qfq`、`hfq` |
| `is_suspended` | bool | `daily_bars`、股票池快照 | 是否停牌或疑似停牌 |
| `data_status` | string | 所有处理后数据 | `ok`、`warning`、`failed`、`stale` |

## 12. 数据质量检查规则

| 规则 | 严重级别 | 检查对象 | 处理建议 |
| --- | --- | --- | --- |
| 必填字段缺失 | error | `trade_date`、`symbol`、`market`、OHLC | 本批数据失败 |
| 主键重复 | error | `trade_date + symbol + market + adjust_type` | 去重前不得进入因子层 |
| 价格关系异常 | error | `high < low`、`high < open/close`、`low > open/close` | 标记异常并剔除 |
| 非正价格 | error | OHLC 小于等于 0 | 标记异常并剔除 |
| 成交额或成交量为 0 | warning | `volume`、`amount` | 可能停牌或流动性不足，进入股票池过滤 |
| 最新交易日滞后 | warning/error | 每个市场最新 `trade_date` | A股超过 1 个预期交易日、港股超过 2 个自然日需告警 |
| 复权视图混用 | error | 同一批 `daily_bars` | 同一任务不得混用 `none`、`qfq`、`hfq` |
| 涨跌幅极端 | warning | `pct_chg` 或 close-to-close 收益 | A股超过涨跌停常识、港股超过阈值需人工复核 |
| 股票池缺元信息 | warning | `securities` | `name`、`exchange`、`status` 缺失时可进入行情层，但需补充 |
| 估值快照缺估值指标 | warning | `valuation_snapshots` | `pe_ttm`、`pb`、`total_market_cap` 全缺时只保留行情字段，不进入估值因子 |
| 交易日历为空 | error | `trade_calendar` | 不能作为调度校验依据 |

## 13. 仍需主控确认

1. 港股交易日历是否接受“用实际行情日期反推”的 MVP 做法，还是要新增独立免费数据源。
2. 港股指数历史行情使用哪个免费源；AkShare 当前可见港股指数快照接口可做状态展示，但稳定历史基准仍需补源。
3. 财务/估值字段是否进入第一版模型输入；数据线程建议先只做估值快照，财报字段等待回测线程确认披露日规则。
4. 股票池过滤阈值，例如最低 20 日平均成交额、最低价格、市值下限。
