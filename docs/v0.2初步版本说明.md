# MM2 v0.2 初步版本说明

## 已完成范围

- 前端页面优先连接本地后端 API，后端不可用时自动回落到本地样例数据。
- 后端新增宏观风控接口，页面会在选股前展示“是否允许观察”的宏观闸门。
- 观察名单改为 SQLite 本地持久化，支持多人登录后按用户隔离。
- 新增 `scripts/run_strategy_sample.py`，可从 `data/processed/` 的真实免费数据样本生成因子、策略信号和 Top-N 榜单。
- 榜单 API 会优先读取 `data/processed/rankings/` 里的本地策略输出，读不到时再回落到内置样例榜单。
- 项目版本号升级到 `0.2.0`，本地后端健康检查会返回 v0.2 版本信息。

## 本地启动顺序

1. 启动后端：

```text
.venv/bin/uvicorn backend.api.app:app --host 127.0.0.1 --port 8000
```

2. 启动前端：

```text
cd frontend
node server.mjs
```

3. 访问页面：

```text
http://localhost:5173/
```

## 策略链路样本

重新生成本地策略样本：

```text
.venv/bin/python scripts/run_strategy_sample.py
```

生成结果会写入：

```text
data/processed/factors/
data/processed/strategy_signals/
data/processed/rankings/
```
